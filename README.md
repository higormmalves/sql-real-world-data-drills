# [PT-BR] 📊 Maven Analytics: Real-World Data Drills (SQL)

## 📖 Sobre o Repositório
Este repositório documenta a minha resolução dos **Real-World Data Drills** propostos pela [Maven Analytics](https://mavenanalytics.io/data-drills). O objetivo principal deste projeto é demonstrar a aplicação prática de **SQL** na resolução de problemas complexos de negócios.

Como um profissional focado em dados, utilizo estes desafios para consolidar técnicas avançadas de manipulação, limpeza, otimização de consultas e modelagem de dados — habilidades essenciais para extrair insights precisos e apoiar a tomada de decisão no ambiente corporativo.

## 🛠️ Tecnologias e Habilidades Aplicadas
- **Linguagem:** SQL (Scripts compatíveis com os principais SGBDs)
- **Técnicas em Destaque:** 
  - Window Functions (LAG, LEAD, Rolling Averages)
  - Common Table Expressions (CTEs) Recursivas
  - Self-Joins e Non-Equi Joins
  - Lógica de Slowly Changing Dimensions (SCD)
  - Manipulação e achatamento de estruturas JSON
  - Limpeza, desduplicação e imputação de dados nulos
  - Market Basket Analysis (Análise de Afinidade)

## 🚀 Índice de Desafios

Os exercícios estão organizados abaixo por nível de dificuldade, detalhando o contexto de negócio e as principais técnicas de SQL praticadas em cada um.

| 🧩 Desafio | 🎯 Habilidades Técnicas Praticadas |
| :--- | :--- |
| **Rolling Up, Looking Back** | Agrupamentos e análise de variação mês a mês (MoM). |
| **Final Form** | Desduplicação de registros, priorizando a linha mais recente. |
| **Making the Cut** | Categorização de dados (*binning*) e cálculos de intervalos de tempo. |
| **Turning Bullish** | Cálculos móveis (*rolling averages*) e criação de *flags* condicionais. |
| **Readmission Radar** | Janelas de tempo móveis e pareamento de eventos. |
| **Splitting the Field** | Agregações baseadas em campos de múltiplos valores. |
| **Movie Metrics** | Engenharia de *features* usando lógica condicional e agregações. |
| **Flatten the Stack** | Transformação de colunas JSON aninhadas para formato tabular. |
| **Estimate the Estate** | Preenchimento de nulos (*imputation*) com médias e lógica condicional. |
| **Spot the Sale** | *Joins* não tradicionais baseados em intervalos de datas. |
| **Booking Breakdown** | Expansão de intervalos de datas e filtragem avançada. |
| **The Price is Right** | Técnicas de *Slowly Changing Dimensions* (SCD). |
| **Streak Leaderboard** | Comparações de datas, funções LAG e lógica de sequências (*streaks*). |
| **Cart Combos** | *Market Basket Analysis* (Análise de afinidade) e *Self-joins*. |
| **Org Chart Overhaul** | Lógica recursiva (CTEs) e modelagem de hierarquias organizacionais. |

## 📁 Estrutura do Projeto
Cada desafio possui sua própria pasta contendo:
* O script SQL contendo a solução (`solution.sql`).
* Um breve arquivo Markdown explicando o raciocínio utilizado na elaboração da *query*.

# [EN] 📊 Maven Analytics: Real-World Data Drills (SQL)

## 📖 About the Repository
This repository documents my solutions to the **Real-World Data Drills** provided by [Maven Analytics](https://mavenanalytics.io/data-drills). The main goal of this project is to demonstrate the practical application of **SQL** in solving complex business problems.

As a data-focused professional, I use these challenges to consolidate advanced techniques in data manipulation, cleaning, query optimization, and data modeling—essential skills for extracting accurate insights and supporting decision-making in a corporate environment.

## 🛠️ Technologies and Applied Skills
- **Language:** SQL (Scripts compatible with major RDBMS)
- **Highlighted Techniques:** 
  - Window Functions (LAG, LEAD, Rolling Averages)
  - Recursive Common Table Expressions (CTEs)
  - Self-Joins and Non-Equi Joins
  - Slowly Changing Dimensions (SCD) Logic
  - JSON structure manipulation and flattening
  - Data cleaning, deduplication, and null value imputation
  - Market Basket Analysis (Affinity Analysis)

## 🚀 Challenge Index

The exercises are organized below, detailing the business context and the main SQL techniques practiced in each.

| 🧩 Challenge | 🎯 Technical Skills Practiced |
| :--- | :--- |
| **Rolling Up, Looking Back** | Groupings and month-over-month (MoM) variance analysis. |
| **Final Form** | Record deduplication, prioritizing the most recent row. |
| **Making the Cut** | Data binning and time interval calculations. |
| **Turning Bullish** | Rolling averages and conditional flags creation. |
| **Readmission Radar** | Rolling time windows and event pairing. |
| **Splitting the Field** | Aggregations based on multi-value fields. |
| **Movie Metrics** | Feature engineering using conditional logic and aggregations. |
| **Flatten the Stack** | Transforming nested JSON columns into a tabular format. |
| **Estimate the Estate** | Null imputation using averages and conditional logic. |
| **Spot the Sale** | Non-traditional joins based on date ranges (Non-Equi Joins). |
| **Booking Breakdown** | Date range expansion and advanced filtering. |
| **The Price is Right** | Slowly Changing Dimensions (SCD) techniques. |
| **Streak Leaderboard** | Date comparisons, LAG functions, and streak logic. |
| **Cart Combos** | Market Basket Analysis (Affinity Analysis) and Self-joins. |
| **Org Chart Overhaul** | Recursive logic (CTEs) and organizational hierarchy modeling. |

## 📁 Project Structure
Each challenge has its own folder containing:
* The SQL script with the solution (`solution.sql`).
* A brief Markdown file explaining the reasoning behind the query.
